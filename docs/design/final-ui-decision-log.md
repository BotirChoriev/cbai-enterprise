# CBAI Final UI Decision Log

Three-option evaluation (1–10). **Winner** selected for platform-wide coherence.

Scoring axes: clarity, IOS identity, evidence/trust, a11y, mobile, consistency, density, performance, risk, scalability, originality, learning cost.

---

## 1. Global shell

| Option | Description | Avg | Winner |
|--------|-------------|-----|--------|
| A | Conservative flat admin shell | 6.2 | |
| B | Spatial navy layered canvas + glass surfaces | **8.7** | **✓** |
| C | Operational workflow-first dense shell | 7.4 | |

**Choice: B** — Semantic tokens (`--cbai-canvas`, `--cbai-topbar-bg`, `--cbai-nav-*`) with warm mineral light variant.
